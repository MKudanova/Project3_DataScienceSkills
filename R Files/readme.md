## R Files

This folder contains all R Quarto (`.qmd`) files used throughout the project.

### Files Included
- `data_transformation.qmd`
  Includes code for data cleaning and tranformation.  
- `dbload_validation.qmd`  
  Includes code for connecting to the PostgreSQL database and performing validation checks to ensure data is loaded correctly.
